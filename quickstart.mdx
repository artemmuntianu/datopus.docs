Getting started with sending events using the JavaScript tracker is very similar to other web analytics vendors like Google Analytics, Adobe Analytics, etc.

The process involves the following high level steps:

#### Configuring a tracker
Configure an instance of the tracker by calling `newTracker` with your desired properties.

```javascript javascript
window.datopus('newTracker', 'www.tracked_domain.com')
```

where `www.tracked_domain.com` is one of domains you specified in the Datopus Dashboard.

<Tip>Tip for B2B: instead of hardcoding a domain name, obtain it from a browser variable `window.location.hostname`</Tip>

#### Tracking a feature
Use the track method `trackFeature` to register activity on a feature.

	- For example, you can register engagement on feature `CTA: Add To Cart` with next code

	```javascript javascript
	window.datopus('trackFeature', 'CTA: Add To Cart');
	```

	- Or, you can add some context

	```javascript javascript
	window.datopus('trackFeature', 'CTA: Add To Cart', {
		user_subscription: 'Premium',
		product_name: 'Product A',
		product_category: 'Category B'
	});
	```
